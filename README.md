# 📝 To-Do List

A simple and clean To-Do List web application built using **HTML**, **CSS**, and **JavaScript**. Add, manage, and keep track of your daily tasks right in the browser.

## Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ☑️ Mark tasks as complete
- 💾 Persists tasks using local storage

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

### Prerequisites

You just need a web browser. No installations or dependencies required.

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. Navigate into the project folder
   ```bash
   cd your-repo-name
   ```
3. Open `index.html` in your browser
   ```bash
   open index.html
   ```
   (or just double-click the file)

## Usage

1. Type your task into the input field.
2. Click **Add** (or press Enter) to add it to the list.
3. Click the checkbox to mark a task as complete.
4. Click the delete icon to remove a task.
